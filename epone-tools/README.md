# EP One Newcomer Tools

Terminal checklist installer for EP One newcomers.

## Install

```sh
./install.sh
```

Then run:

```sh
epone
```

## Features

- Dev wiki Markdown menu
- Manual setup Markdown menu
- Program checklist with host/project-shell scopes
- Select all with `a`
- Safe mode by default
- Yolo mode for direct install
- macOS, Linux, and WSL Ubuntu/Debian support

## Edit Menu Content

- `resources/dev-resource.txt`
- `resources/dev-wiki.md`
- `resources/manual-setup.md`
- `resources/required-programs.txt`
